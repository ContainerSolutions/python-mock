# python-mock
Unit testing and mocking in Python

## How to run a test

Without a mock:

```
$ ./test_mymodule.py
.
----------------------------------------------------------------------
Ran 1 test in 0.000s

OK
```

With mock:

```
$ ./test_mymodule_mock.py
.
----------------------------------------------------------------------
Ran 1 test in 0.000s

OK
```
